# Comprehensive Healthcare Analytics Dashboard using Power BI

## 📌 Project Overview
This project focuses on building an interactive Healthcare Analytics Dashboard using Microsoft Power BI. The dashboard transforms raw healthcare data into meaningful visual insights related to patients, doctors, appointments, and hospital operations.

The system helps healthcare administrators and analysts monitor patient trends, doctor performance, consultation efficiency, and operational activities through dynamic visualizations and KPI tracking.

---

## 🚀 Features

- Interactive Power BI Dashboard
- Patient Demographics Analysis
- Doctor Performance Monitoring
- Appointment & Consultation Tracking
- KPI Cards for Key Metrics
- Dynamic Filters and Slicers
- Healthcare Trend Analysis
- Region-wise and PHC-wise Insights
- Real-time Data Exploration

## ⚙️ Data Processing

### Data Cleaning
- Removed duplicate values
- Handled missing data
- Standardized columns and formats

### Data Modeling
- Created relationships between tables
- Implemented star schema structure

### DAX Measures
Some key measures include:

```DAX
Total Patients = COUNT(Patient_ID)

Total Appointments = COUNT(Appointment_ID)

Total Doctors = DISTINCTCOUNT(Doctor_ID)
```
